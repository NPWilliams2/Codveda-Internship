# Basic Calculator

## Description 
A console-based C++ calculator created for the Codveda C/C++ Development Internship.

The program allows users to perform basic arithmetic operations and repeat calculations.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Division-by-zero handling
- Input validation for numbers
- Repeat calculation option

## Technologies Used
- C++
- Visual Studio
- Git
- GitHub
