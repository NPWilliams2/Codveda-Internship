# Number Guessing Game

## Description
A console based Number Guessing Game developed in C++ as part of the codveda C/C++ Development Internship

The program generates a random number between 1 and 100, and the player must guess the correct number. The game provides feedback after each guess and tracks the total number of attemps required to find the correct answer.

## Features
- Random number generation using rand()
- Multiple guessing attemps until the correct answer is found
- Feedback for guesses that are too high or too low
- Input validation for non-numeric entries
- Range validation (1-100)
- Attempt counter
- Play-again functionality

## Technologies Used
- C++
- Visual Studio

## Concepts Demonstrated
- Random number generation
- Loops
- Conditional Statements
- Input validation
